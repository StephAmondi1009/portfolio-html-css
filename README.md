# Portfolio HTML/CSS

A simple one‑page personal portfolio built with HTML and CSS.  
Created to practice Git, GitHub, VS Code, and deployment with GitHub Pages.

## Sections

- Home: Short introduction and call‑to‑action.
- About: Brief description of background and interests.
- Projects: Small list of example projects.
- Contact: Email and GitHub links.

## How to run

1. Clone the repository:
   - In VS Code: use the Git clone command or `git clone` in the terminal.
2. Open the folder in VS Code.
3. Open `index.html` in a browser (you can use a Live Server extension if you like).

## Deployment

This project can be deployed using GitHub Pages:
- Set the source branch to `main` in the repository settings.
- After a few minutes, the site will be available at `https://<your-username>.github.io/<repo-name>/`.
